# Learning C Programming

A structured guide to learning C programming from fundamentals to advanced concepts.

## Contents

| Chapter | Topic | Description |
|---------|-------|-------------|
| Ch-1 | Header Files | C header files and reserved keywords |
| Ch-2 | Hello World | First C program and code comments |
| Ch-3 | Variables | Variable declaration, data types, and constants |
| Ch-4 | Input/Output | Reading and writing data with scanf/printf |
| Ch-5 | Operators | Arithmetic, relational, logical, and bitwise operators |
| Ch-6 | Decision Making | Conditional statements and ternary operator |
| Ch-7 | Loops | Loop constructs (for, while, do-while) |
| Ch-7A | Patterns | Pattern printing with nested loops |
| Ch-8 to Ch-22 | Coming Soon | Functions, recursion, arrays, strings, pointers, memory management, file handling |

## Setup

### Requirements
- C compiler (GCC, Clang, or MSVC)
- Text editor or IDE

### Compilation
```bash
gcc filename.c -o filename
./filename              # Linux/macOS
filename.exe            # Windows
```

## Learning Structure

Each chapter contains executable C programs with accompanying documentation explaining the concepts, syntax, and usage patterns.

## Quick Reference

### Data Types
| Type | Size |
|------|------|
| char | 1 byte |
| int | 4 bytes |
| float | 4 bytes |
| double | 8 bytes |

### Format Specifiers
| Specifier | Use |
|-----------|-----|
| %d | Integer |
| %f | Float |
| %c | Character |
| %s | String |

### Key Operators
- Arithmetic: `+`, `-`, `*`, `/`, `%`
- Relational: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Logical: `&&`, `||`, `!`
- Bitwise: `&`, `|`, `^`, `~`, `<<`, `>>`
